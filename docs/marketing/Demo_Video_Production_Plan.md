# Demo Video Production Plan

**Total Videos:** 16
**Purpose:** App walkthrough demo recordings with voiceover — for onboarding tooltips and marketing.

**Recording notes:**
- Reset walkthrough before each video that shows tooltips: Settings → Replay App Walkthrough
- Use a clean demo account with sample data pre-loaded
- Enable Do Not Disturb before recording
- Record at 1080p minimum
- Tooltip text shown on screen is in **[brackets]** in the voiceover — narrator reads these aloud as they appear

## Recording Checklist

- [ ] Use a clean demo account (no personal data visible)
- [ ] Reset walkthrough before each video that shows tooltips: **Settings → Replay App Walkthrough**
- [ ] Enable **Do Not Disturb** before recording
- [ ] Set device to target language for each video
- [ ] Record at **1080p** minimum, consistent zoom/font size across all videos
- [ ] Each video is self-contained — start from the relevant screen, not mid-flow
- [ ] Record English voiceover first; Hindi/Malayalam can be dubbed separately
- [ ] Upload to **YouTube as Unlisted** → paste URLs into `WalkthroughVideoConfig`
