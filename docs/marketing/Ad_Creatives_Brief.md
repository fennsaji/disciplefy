## A/B Testing Plan

### Test 1: Creative Format
- Version A: Video ad (15 sec)
- Version B: Carousel ad
- Version C: Single image ad
- Metric: Install Rate, CPI

### Test 2: Headline Messaging
- Version A: Feature-focused ("AI Bible Study")
- Version B: Benefit-focused ("Understand the Bible")
- Version C: Social proof ("10,000+ Users")
- Metric: CTR, Install Rate

### Test 3: Language
- Version A: English only
- Version B: Hindi only
- Version C: Mixed (Hindi + English)
- Metric: CTR, CPI by region

### Test 4: CTA Button
- Version A: "Download"
- Version B: "Install Now"
- Version C: "Get Started"
- Version D: "Learn More"
- Metric: CTR

### Test 5: Audience
- Version A: Broad Christian interest
- Version B: Specific denomination interest
- Version C: Lookalike audiences
- Metric: CPI, Quality of installs (retention)
