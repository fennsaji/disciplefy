# Demo Video Production Plan — Hindi Voiceover

**Total Videos:** 16
**Language:** Hindi (हिन्दी)
**Purpose:** App walkthrough demo recordings with Hindi voiceover.

**Recording notes:**
- Reset walkthrough before each video that shows tooltips: Settings → Replay App Walkthrough
- Use a clean demo account with sample data pre-loaded
- Enable Do Not Disturb before recording
- Record at 1080p minimum
- Tooltip text shown on screen is in **[brackets]** in the voiceover — narrator reads these aloud as they appear
